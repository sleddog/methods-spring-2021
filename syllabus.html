Methods 2021 - Syllabus

Course Syllabus – CSCI 494, Industry Methods
1. Course number and name: CSCI 494, Industry Methods
2. Credits and contact hours: 1 credit, 1 hour (lecture)
3. Instructor’s or course coordinator’s name: Devin Gray
4. Text book, title, author, and year: none
• Other supplemental materials: none
5. Specific course information
• Course Catalog Description: The seminar will introduce you to various industry standards such as git, docker, kubernetes, CI (continuous integration), JIRA, IDE setup, and more.
• Prerequisites or Co-requisites: Preq: CSCI 232
• Required, Elective, or Selected elective (as per Table 5-1): Elective
6. Specific goals for the course
• Course Outcomes:
i. Be comfortable with using / interacting with a code repository (i.e. git)
ii. Understand concepts of continuous delivery and continuous integration
iii. Deploy and test software to a cloud environment
iv. Understand concepts involving telemetry and application monitoring
• Criterion 3 Student Outcomes Addressed: A & I

7. Brief list of topics to be covered
• Git
• Shell Scripting
• Testing
• Code Review
• Web Applications
• API
• Docker
• Software deployment methods
• Kubemeters
• Telemetry
• Frontend Engineering
