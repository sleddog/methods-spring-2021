# FizzBuzz in Python
## Overview
This is a shell script that runs an implementation in python. 
## Instructions
To run using "./run.sh" you will likely need to change the file permission to executable.
This can be done using 
''' sh
chmod 755 run.sh
'''

You can also use the "sh" command.
''' sh
sh run.sh
'''
